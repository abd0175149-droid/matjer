# 03 — ميزات المتجر (Storefront Features)

## 1. الصفحة الرئيسية

- بانر رئيسي (Slider) للعروض والمنتجات المميزة.
- أقسام التصنيفات الرئيسية (أطقم، خواتم، أساور، قلادات).
- قسم "الأكثر مبيعاً".
- قسم "وصل حديثاً".
- قسم العروض والخصومات.
- شريط مزايا (شحن، ضمان، دفع آمن، إرجاع).
- تذييل (Footer): روابط، تواصل، سياسات، وسائل تواصل اجتماعي.

---

## 2. التصفح والتصنيفات

- تصنيفات رئيسية وفرعية.
- تصنيف حسب النوع: **ذهب روسي / ذهب صيني**.
- عرض شبكي (Grid) مع صور واضحة.
- ترتيب: الأحدث، السعر تصاعدي/تنازلي، الأكثر مبيعاً، الأعلى تقييماً.
- ترقيم الصفحات أو التمرير اللانهائي (Infinite Scroll).

### الفلترة (Filters)
| الفلتر | القيم |
|--------|-------|
| النوع | روسي / صيني |
| التصنيف | أطقم / خواتم / أساور / قلادات |
| السعر | نطاق سعري (من - إلى) |
| اللون | حسب المتوفر |
| المقاس | للخواتم والأساور |
| التوفّر | متوفر فقط |

---

## 3. البحث

- شريط بحث فوري مع اقتراحات أثناء الكتابة (Autocomplete).
- بحث بالاسم، التصنيف، الكود (SKU).
- تصحيح الأخطاء الإملائية واقتراح البدائل.
- نتائج بحث قابلة للفلترة والترتيب.

---

## 4. صفحة المنتج

### العرض
- معرض صور متعدد (Gallery) مع خاصية التكبير (Zoom).
- فيديو للمنتج (اختياري) وعرض 360° (اختياري).
- اسم المنتج، الكود (SKU)، التصنيف، النوع.
- السعر (والسعر قبل الخصم إن وُجد).
- حالة التوفّر والكمية المتاحة.

### التفاصيل
- وصف المنتج.
- المواصفات (اللون، الخامة، القياسات، عدد القطع للأطقم).
- اختيار المتغيرات (المقاس / اللون) قبل الإضافة للسلة.
- اختيار الكمية.

### التفاعل
- زر "أضف إلى السلة".
- زر "أضف إلى المفضلة".
- مشاركة المنتج (واتساب / تواصل اجتماعي / نسخ الرابط).
- قسم التقييمات والمراجعات.
- قسم "منتجات مشابهة" و"قد يعجبك أيضاً".

---

## 5. السلة (Cart)

- عرض المنتجات المضافة بصورها وأسعارها.
- تعديل الكمية أو حذف منتج.
- حساب المجموع الفرعي تلقائياً.
- إدخال كوبون الخصم.
- عرض رسوم الشحن التقديرية.
- زر المتابعة للدفع.
- حفظ السلة للعميل المسجّل (Persistent Cart).

---

## 6. الدفع (Checkout)

- ملخص الطلب النهائي.
- اختيار/إضافة عنوان التوصيل.
- اختيار طريقة الشحن.
- اختيار طريقة الدفع:
  - دفع إلكتروني (بطاقة / بوابة محلية).
  - الدفع عند الاستلام (COD).
- تطبيق الكوبونات والخصومات.
- تأكيد الطلب واستلام رقم الطلب.
- دعم الشراء كضيف (Guest Checkout) دون تسجيل.

---

## 7. حساب العميل

| القسم | الوظيفة |
|-------|---------|
| الملف الشخصي | تعديل البيانات وكلمة المرور |
| العناوين | إضافة/تعديل/حذف عناوين متعددة |
| طلباتي | سجل الطلبات وحالاتها |
| تتبّع الطلب | متابعة حالة الطلب لحظياً |
| المفضلة | المنتجات المحفوظة |
| المراجعات | تقييماتي السابقة |

### التسجيل والدخول
- تسجيل بالبريد أو رقم الهاتف.
- دخول عبر Google (اختياري).
- استعادة كلمة المرور.
- تحقق عبر OTP (رسالة نصية / بريد).

---

## 8. مزايا الثقة والتفاعل

- **التقييمات والمراجعات**: نجوم + تعليق + صور من العملاء.
- **شارات الثقة**: ضمان، دفع آمن، إرجاع سهل.
- **الدعم المباشر**: شات أو زر واتساب عائم.
- **الإشعارات**: بريد/SMS لتأكيد الطلب وتحديثات الحالة.
- **سياسات واضحة**: الإرجاع، الاستبدال، الضمان، الشحن.

---

## 9. مزايا تسويقية

- نظام كوبونات وأكواد خصم.
- عروض (خصم نسبة / مبلغ / اشترِ X واحصل على Y).
- قائمة "وصل حديثاً" و"الأكثر مبيعاً" تلقائية.
- النشرة البريدية (Newsletter).
- منتجات مقترحة بناءً على التصفح.
- مشاركة اجتماعية لكل منتج.

---

## 10. تجربة المستخدم (UX)

- تصميم متجاوب كامل (جوال / تابلت / حاسوب).
- سرعة تحميل عالية.
- واجهة عربية (مع إمكانية إضافة الإنجليزية).
- دعم الاتجاه من اليمين لليسار (RTL).
- تصميم فاخر يليق بمنتجات الذهب (ألوان: ذهبي/أسود/أبيض).
- سهولة الوصول (Accessibility).
